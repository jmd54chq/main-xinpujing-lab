# main-xinpujing-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常是一些实验性质的小工具、展示页面或静态原型，不针对任何特定网站或单个域名。

## 项目简介

这是一个轻量级的 HTML 页面集合仓库，主要目的是方便维护和分享一些独立的网页文件。每个页面均为自包含的 HTML 文件，不依赖外部数据库或后端服务。适合用于快速展示、测试或存档。

## 目录说明

```
.
├── index.html          # 仓库入口页面（如有）
├── pages/              # 存放各个独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源文件（如样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下存放各独立页面，每个页面均可单独通过浏览器打开。
- `assets/` 目录存放页面可能共享的资源文件，便于统一管理。

## 页面归档说明

本仓库内的 HTML 页面均为独立归档，不相互依赖。每个页面可以独立运行，并可能包含自身的样式与脚本。页面内容可能涉及实验性功能、临时展示或学习笔记，不代表最终产品。

部分页面可能处于未完成或测试状态，请自行评估使用。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进页面或修复问题。
- 新增页面请放入 `pages/` 目录，并保持文件命名清晰。
- 公共资源请尽量放入 `assets/` 目录，避免重复引入。
- 本仓库不保证所有页面长期可用或更新，但会尽力保持基本可读性。

## 使用方式

你可以直接通过 GitHub Pages 或其他静态托管服务部署本仓库，也可以将页面下载到本地使用。每个 HTML 文件均可以直接在浏览器中打开。
